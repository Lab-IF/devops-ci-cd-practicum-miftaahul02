1. Refleksi Pribadi
    * Apa harapan Anda dari praktikum DevOps ini?
      Melalui praktikum DevOps ini, saya memiliki harapan besar untuk tidak hanya sekadar mempelajari teori di kelas, tetapi juga mendapatkan pengalaman langsung ( hands-on) tentang standar industri perangkat lunak modern. Saya ingin mengubah pola pikir saya dari yang awalnya hanya berfokus pada penulisan kode ( coding), menjadi pemahaman menyeluruh tentang siklus hidup perangkat lunak ( Software Development Life Cycle). Saya berharap praktikum ini akan memberikan fondasi yang kuat mengenai konsep otomatisasi dan integrasi, sehingga saya mengerti betul bagaimana proses perangkat lunak yang kompleks dikirimkan dari komputer lokal pengembang hingga dapat diakses oleh pengguna akhir di seluruh dunia secara online tanpa kendala.
    * Skill apa yang ingin Anda kuasai di akhir semester?
      Pada akhir semester, ada beberapa skill krusial yang sangat ingin saya kuasai secara praktis. Pertama, saya ingin mahir dalam menggunakan Git/GitHub untuk kolaborasi tim yang lancar tanpa adanya konflik kode yang merusak. Dan saya juga menargetkan untuk mampu menyusun rancangan otomatisasi dasar menggunakan CI/CD Pipeline, sehingga proses testing dan deployment aplikasi dapat berjalan secara otomatis ketika ada pembaruan kode.
